Mobile websites

Situation: Freerecordshop, Eci en Cosmox were not designed with mobile in mind. It was almost impossible to order on your phone. That had to be changed!

Task: make a mobile design and implement this for mobile users. Once all the necessary functionalities are built the resonsive mobile design is to replace the current desktop versions.

Action: turn the desktop design into a (responsive) mobile design. Implement this using parts of existing css frameworks (sass).

Result: a new mobile designe for eci.nl / eci.be.
